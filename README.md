# Project Overview

The **Global Sports Footwear Sales Dashboard** provides a comprehensive view of sales performance using multiple visual analytics components.

The dashboard allows users to:

- Monitor **total sales, discounts, and units sold**
- Analyze **brand performance**
- Track **sales trends by year**
- Understand **customer ratings**
- Compare **sales by category, model, size, and color**
- Explore **global sales distribution by country**
- Evaluate **payment method usage**
